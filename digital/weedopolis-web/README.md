# Weedopolis Web Prototype

Playable browser prototype for Weedopolis: Strain City Edition.

Open index.html in a browser or serve this folder with a static server.

Included:
- 40-space Weedopolis board
- 2-8 local players
- Custom player names
- Dice movement
- Buy, auction, rent, tax, utility, category rent, upgrades, mortgage, Trim Jail, cards, and local save/load
- Browser test harness via runWeedopolisTests()

Keep this prototype aligned with:
- ../../data/board_map.csv
- ../../data/game_config.json
- ../../data/rent_ladder.csv
- ../../docs/WEEDOPOLIS_PRODUCTION_BIBLE.md
