# Repository Routing

## Authoritative production repository

All Weedopolis production work must happen in:

`dtfgenetics/Weedopolis-strain-Edition`

## Deleted legacy repository

The older Potopoly repository has been deleted and must not be used for Weedopolis production work.

## Reason

The public-facing game name is locked as **Weedopolis: Strain City Edition**. Using one authoritative repository prevents asset drift, duplicate board maps, wrong file paths, and outdated naming.

## Rule for future work

All new files, prompts, board data, card data, print assets, source files, and digital implementation files must be created in this repository unless the user explicitly says otherwise.
