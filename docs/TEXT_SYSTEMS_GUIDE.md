# Weedopolis Text Systems Guide

## Purpose

This file controls the writing system for Weedopolis so every board space, card, deed, rule, and money piece feels like one game.

The goal is clear and concise tabletop text. Humor is allowed, but function comes first.

## Voice rules

Weedopolis writing should be:

- Short
- Direct
- Playful
- Easy to understand in one read
- Adult cannabis themed without being messy
- Consistent across every component

Avoid:

- Long joke setups
- Ambiguous rules
- Too much slang
- Overly technical grow language
- Text that only makes sense if explained out loud

## Global terminology

Use these exact terms:

| Use | Do not use |
|---|---|
| Weedopolis | Potopoly |
| Bud Bucks | dollars, cash, money points |
| Grow Tents | houses |
| Dispensaries | hotels |
| High Chance | Chance |
| Community Stash | Community Chest |
| Trim Jail | jail, prison |

## Board-space text system

Board spaces need the shortest text in the game.

### Property board spaces

Format:

```text
[STRAIN NAME]
[PRICE]
BUD BUCKS
```

Example:

```text
BLUE DREAM
200
BUD BUCKS
```

Do not include rent, flavor text, jokes, or descriptions on board property spaces.

### Card draw spaces

Format:

```text
[DECK NAME]
DRAW CARD
```

Examples:

```text
HIGH CHANCE
DRAW CARD
```

```text
COMMUNITY STASH
DRAW CARD
```

### Tax and fee spaces

Format:

```text
[SPACE NAME]
PAY [AMOUNT]
BUD BUCKS
```

Examples:

```text
420 TAX
PAY 200
BUD BUCKS
```

```text
LAB TESTING FEE
PAY 100
BUD BUCKS
```

### Category spaces

Format:

```text
[CATEGORY NAME]
200
BUD BUCKS
```

Examples:

```text
INDICA
200
BUD BUCKS
```

```text
AUTOFLOWER
200
BUD BUCKS
```

### Utility spaces

Format:

```text
[UTILITY NAME]
150
BUD BUCKS
```

Examples:

```text
GROW LIGHTS
150
BUD BUCKS
```

```text
WATER WORKS
150
BUD BUCKS
```

## Deed card text system

Deed cards hold the details that do not fit on the board.

### Strain deed format

```text
[PROPERTY NAME]
Purchase Price: [amount] Bud Bucks

Rent: [amount]
Rent with full set: [amount]
With 1 Grow Tent: [amount]
With 2 Grow Tents: [amount]
With 3 Grow Tents: [amount]
With 4 Grow Tents: [amount]
With Dispensary: [amount]

Grow Tent Cost: [amount] Bud Bucks each
Dispensary Cost: [amount] Bud Bucks + 4 Grow Tents
Stash Value: [amount] Bud Bucks
```

### Category deed format

```text
[CATEGORY NAME]
Purchase Price: 200 Bud Bucks

If 1 category owned: Rent 25
If 2 categories owned: Rent 50
If 3 categories owned: Rent 100
If 4 categories owned: Rent 200

Stash Value: 100 Bud Bucks
```

### Utility deed format

```text
[UTILITY NAME]
Purchase Price: 150 Bud Bucks

If one utility owned:
Rent is 4x dice roll.

If both utilities owned:
Rent is 10x dice roll.

Stash Value: 75 Bud Bucks
```

## Draw card text system

Draw cards must separate flavor from action.

### Preferred format

```text
[CARD TITLE]
[Short flavor sentence.]
[Clear mechanical instruction.]
```

Example:

```text
SECRET SMOKE SESSION
You got invited to the good table.
Advance to Start Session. Collect 200 Bud Bucks.
```

### If space is tight

Use only:

```text
[CARD TITLE]
[Clear mechanical instruction.]
```

Example:

```text
BACK ALLEY SHORTCUT
Move forward 3 spaces.
```

## Card effect language library

Use consistent verbs:

| Action | Preferred wording |
|---|---|
| Gain money | Collect [amount] Bud Bucks. |
| Lose money to bank | Pay [amount] Bud Bucks. |
| Lose money to player | Pay [player/every player] [amount] Bud Bucks. |
| Move to space | Advance to [space name]. |
| Move backward | Move back [number] spaces. |
| Jail movement | Go directly to Trim Jail. Do not collect 200 Bud Bucks. |
| Keepable card | Keep this card until needed or traded. |
| Skip turn | Skip your next turn. |

## Tone examples

Good titles:

- Secret Smoke Session
- Back Alley Shortcut
- Strain Hype
- Found Jar
- Community Tip Jar
- Glass Accident
- Grow Tent Sale

Bad titles:

- Extremely long joke names
- Names that hide the effect
- Names that sound like a different game system
- Names that require cannabis industry knowledge to understand

## Text length limits

| Asset | Max recommended text |
|---|---:|
| Board property space | 3 lines |
| Board card space | 2 lines |
| Corner space | 2 to 4 lines |
| Deed card | Full rent table allowed |
| Draw card title | 1 to 4 words |
| Draw card effect | 1 to 2 sentences |
| Money bill | Denomination + Bud Bucks + small branding |
| Token | No text unless needed |

## Capitalization rules

Board space names should use uppercase:

```text
BLUE DREAM
HIGH CHANCE
COMMUNITY STASH
```

Card titles may use title case in data files but can be uppercase in final art.

Rulebook text should use normal sentence case.

## Clarity checklist

Before text is approved, check:

- Can a new player understand it without explanation?
- Is the required payment or movement clear?
- Is the player told who receives money?
- Is the exact space name used?
- Does the text match the data file?
- Is the joke secondary to the rule?

If not, rewrite it.
