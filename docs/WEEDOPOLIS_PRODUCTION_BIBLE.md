# Weedopolis: Strain City Edition — Production Bible

## Project identity

**Game name:** Weedopolis  
**Edition:** Strain City Edition  
**Game type:** Cannabis-themed property-trading board game  
**Tone:** Funny, clean, premium, easy to understand  
**Audience:** Adult cannabis consumers and stoner game-night players  

The goal is not a messy weed meme board. The goal is a professional print-ready game system with a consistent board, card decks, money, tokens, and future digital version.

## Locked naming rule

The public-facing game name is **Weedopolis**.

Do not use Potopoly as the game name in board artwork, card decks, rulebook copy, money, box art, digital UI, file names, or player-facing text.

## Legal/parody boundary

Weedopolis should be built as its own cannabis property-trading game. Do not directly copy protected board art, mascots, exact logos, exact card wording, exact deed wording, or trade dress from existing games.

Allowed direction:

- Property-trading mechanics
- 40-space perimeter board structure
- Buy/rent/build gameplay
- Cannabis-themed replacements
- Parody naming

Avoid:

- Exact Monopoly branding or logo treatment
- Existing board-game mascots or directly similar mascots
- Exact card/deed wording from existing games
- Direct recreation of protected board art

## Board structure

The board must have exactly 40 spaces:

| Type | Count |
|---|---:|
| Corners | 4 |
| Strain properties | 22 |
| Category spaces | 4 |
| Card draw spaces | 6 |
| Utilities | 2 |
| Taxes/fees | 2 |
| Total | 40 |

## Orientation rule

All spaces face outward from the center:

| Board side | Facing |
|---|---|
| Bottom row | Bottom edge |
| Left row | Left edge |
| Top row | Top edge |
| Right row | Right edge |

Property tile internal order is always:

1. Color bar
2. Strain name
3. Icon/image
4. Bold price

The whole tile rotates based on its board side.

## Visual direction

Preferred visual style:

- Vintage board game
- Cannabis city map
- Premium dispensary menu
- Cream parchment background
- Deep green ink
- Gold accent details
- Bold readable text
- Clean color bars
- Simple consistent icons

Avoid:

- Cheap clipart
- Inconsistent icon styles
- Tiny unreadable board text
- Over-detailed artwork inside small spaces
- Fake transparency checkerboards
- Random AI-generated whole-board redraws

## Print production rules

Final print assets should be:

- 300 DPI
- Proper physical dimensions
- Bleed versions included
- No-bleed proof versions included
- Editable source versions preserved
- Version-numbered filenames

Board target master size: **20 in × 20 in** unless changed intentionally.

Cards should include fronts, backs, bleed-safe print sheets, and cut guides.

## Build order

1. Lock master board map
2. Build blank board skeleton
3. Build four row templates
4. Build individual space assets
5. Build full board assembly
6. Build deed cards
7. Build High Chance deck
8. Build Community Stash deck
9. Build Bud Bucks
10. Build tokens and upgrade pieces
11. Build rulebook
12. Build box files
13. Build digital game data/assets

## Acceptance checklist

A board file is not accepted unless:

- It has exactly 40 spaces
- It has 4 corners
- All spaces face outward correctly
- It has no missing or duplicate spaces
- Names are readable
- Prices are readable and bold
- Icons use one consistent visual system
- Color bars match the board map
- High Chance and Community Stash spaces are present
- Grow Tents and Dispensaries are represented correctly
- 300 DPI print export exists
- Bleed export exists
- Editable source exists
