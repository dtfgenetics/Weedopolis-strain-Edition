# Weedopolis Board Assets

## Phase 02 asset status

This folder contains the first production-ready source SVG assets for the Weedopolis board system.

## Created assets

### Board base

- `base/blank_board_base_20x20.svg`

Purpose:

- 20 in x 20 in board base
- 6000 x 6000 viewBox
- 40 total perimeter spaces
- 4 corner spaces
- 9 non-corner spaces per side
- Center safe zones
- No final names, prices, or icons

### Template

- `templates/property_tile_template_color_title_image_price.svg`

Locked layout:

1. Color bar
2. Title
3. Main image/icon
4. Bottom price

## Acceptance focus

These are proof assets, not final art. They exist to validate:

- Correct tile hierarchy
- Correct board geometry
- Correct outward-facing side logic
- Correct price placement at the bottom
- Reusable SVG structure

The locked property tile order is:

**Color → Title → Image → Bottom Price**
